高度塌陷

现象

效果：

代码：

html代码

<div class="height-collapse" > <img src="../../assert/images/demo_1.jpg"> </div>

css 代码

.height-collapse { background-color: red; min-height: 20px; } .height-collapse img { float: left; width: 300px; }

解决方案

  1. 效果：

    描述:

    通过::after伪类在高度塌陷元素的最后插入一个伪元素，并设置这该元素清除浮动，但是由于clear只有对块级元素才有效，而::after伪元素默认都是内联水平，所以还要将伪元素的display设置为block，这样就可以达到效果了。

    代码：

    html代码

    <div class="height-collapse-fix-1" > <img src="../../assert/images/demo_1.jpg"> </div>

    css 代码

    .height-collapse-fix-1 { background-color: red; min-height: 20px; } .height-collapse-fix-1 img { float: left; width: 300px; } .height-collapse-fix-1::after { content: ""; display: block; clear: both; }

  2. 效果：

    描述:

    触发元素的BFC特性，比如将overflow设置为hidden，BFC具体介绍请戳这里

    代码：

    html代码

    <div class="height-collapse-fix-2" > <img src="../../assert/images/demo_1.jpg"> </div>

    css 代码

    .height-collapse-fix-2 { background-color: red; min-height: 20px; overflow: hidden; } .height-collapse-fix-2 img { float: left; width: 300px; }
