overflow:hidden，子元素超出父元素后，父元素padding失效

现象

效果：

代码：

html代码

<div class="parent-elm-effort"> <img class="child-elm-effort" src="../../assert/images/demo_1.jpg"> </div>

css 代码

.parent-elm-effort { width: 300px; height: 160px; padding: 10px; overflow: hidden; background-color: blue; }

解决方案

  1. 效果：

    描述:

    通过透明边框来实现padding效果

    代码：

    html代码

    <div class="parent-elm-resolve"> <img class="child-elm-resolve" src="../../assert/images/demo_1.jpg"> </div>

    css 代码

    .parent-elm-resolve { width: 300px; height: 160px; /* padding: 10px; */ border: 10px solid transparent; overflow: hidden; background-color: lightblue; }
